A cross-platform Socks5/Http/SS/SSR/Vmess/Trojan(go)/Snell GUI client


## Feature
- Full PAC Control
- Cross-platform
- User Rule Support
- Http Support
- Subscription Support
- Import Servers from many methods
- Mutiple servers Support
- Socks5/Http/SS/SSR/Vmess/Trojan(go)/Snell Support
- ....and more
